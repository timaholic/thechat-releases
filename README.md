# The Chat — release downloads

Binaries for **The Chat**, a voice-to-text intercom for live production crews.

This repository hosts release assets only — it holds no source. The Sparkle
update feed lives separately at <https://gainware.io/thechat/appcast.xml> and
carries the EdDSA signature that authenticates every download here, so the
split hosting is safe: a tampered asset fails signature validation.

Releases are published by `publish.sh`.
